What

Grunt.js is a build tool written in JavaScript built on Node.js.

Why

Used by these folks

  • Twitter
  • jQuery
  • jQuery UI
  • Modernizr
  • Adobe

Why

  • Focused on common front-end tasks
  • Configuration is just JavaScript
  • Cross platform - works on OS X / Windows / Linux
  • Leverage the power of package.json and npm

Why

Installing your project should be as simple as:

      $ git clone yourRepo newProject
      $ cd newProject
      $ npm install // Install all dependencies
      $ grunt       // Concat, minify, run tests, 
                    // deploy to CDN, etc

Installation

Old and busted

      $ npm install -g grunt

New hotness

      $ npm install -g grunt-cli
      $ npm install grunt // In each project

Installation

Install the latest version of grunt and save the requirement to package.json

      $ npm install grunt --save-dev
Grunt is installed to: 
      ./node_modules/grunt/lib/grunt.js

Grunt Basics
Demo

Concepts: Multi-task

      "task": {
        "options": { /* options for all tasks */ }
        "foo": {",
          "options": { /* override the global options */ }
        "},"
        "bar": { /* other task */ }
      }
      grunt.registerTask('default', [ 'task:foo' , 'task:bar' ]);

Concepts: File lists

      files: ['src/aa.js', 'src/aaa.js']
    
      src: ['src/bb.js', 'src/bbb.js'],
      dest: 'dest/b.js',
    
      files: {
        'dest/a.js': ['src/aa.js', 'src/aaa.js'],
        'dest/a1.js': ['src/aa1.js', 'src/aaa1.js'],
      }

Concepts: Globbing

Match all files in directory foo 
      'foo/*'
Match all javascript files in any subdirectory of foo 
      'foo/**/*.js'
    
      grunt.file.expand(['styles/**/*', 'js/**/*.js'])

Concepts: Globbing and File lists

      files: ['app/**/*.js', 'vendor/**/*.js']
    
      src: ['app/**/*.js', 'vendor/**/*.js'],
      dest: 'dest/b.js',
    
      files: {
        'dest/a.js': ['app/**/*.js', 'vendor/**/*.js'],
      }

Concepts: Read other files

Read file contents synchonously 
      grunt.file.read('foo')
Read and parse JSON file 
      grunt.file.readJSON('foo.json')
    
      password: grunt.file.read('password.txt')
      pkg: grunt.file.readJSON('package.json')

Concepts: Templates

Grunt expands text passed in using underscore templates. 
      pkg: grunt.file.readJSON('package.json'),
      config: {
        outputPath: 'foo/bar',
      }, /* ... */
      copy: {
        dst: '<%= config.outputPath %>/app_<%= pkg.version %>.js'
      }
